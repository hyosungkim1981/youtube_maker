# subtitle-generator 스킬 (개요 버전)

## 1. 역할

- `script_{n}.md`를 기준으로 시간 정보를 가진 자막 텍스트를 생성한다.
- (옵션) 실제 오디오/영상 길이 정보를 활용해 타임스탬프를 정교하게 조정한다.

## 2. 입력

- `/output/{date}/script_{n}.md` 경로
- (옵션) `/output/{date}/video_{n}.mp4` 또는 오디오 파일 경로

## 3. 출력

- `/output/{date}/subtitles_{n}.srt` (또는 `.vtt`)

## 4. 성공 기준

- 자막 파일 포맷이 유효하다 (SRT/VTT 문법 검증).
- 영상/오디오가 있다면, 전체 길이와 자막의 마지막 타임스탬프가 일정 범위 내에서 일치한다 (예: ±5초).

## 5. 검증과 실패 처리

- **검증**
  - 파일 존재 여부 및 포맷 파싱 테스트
  - (가능하다면) 영상/오디오 길이와의 차이 계산
- **실패 처리**
  - 포맷 오류 → 자동 재생성(최대 2회), 타임스탬프를 더 넉넉하게 재배분.
  - 여전히 맞지 않으면, 자막 파일을 “대략적인 가이드”로 표시하고, 사람이 검수 단계에서 조정하도록 안내.
