# script-writer 스킬 (개요 버전)

> NOTE: 실제 문체/톤/예시는 구현 시점에 채운다.

## 1. 역할

- `topics.json`의 각 주제에 대해 **5분 분량**의 영상 스크립트 초안을 생성한다.
- 도입–전개–정리 구조를 갖춘 자연스러운 흐름을 만든다.
- 채널 고유 톤/브랜드 가이드가 있다면 이를 반영한다.

## 2. 입력

- `/output/{date}/topics.json` 경로
- (옵션) `/docs/channel-style-guide.md` 경로
- 대상 topic id 또는 인덱스 (주제 1개 단위로 호출)

## 3. 출력

- `/output/{date}/script_{n}.md`

스크립트는 대략 아래와 같은 섹션 구조를 갖는다 (예시):

```markdown
# [영상 제목 후보]

## 1. 오프닝
...

## 2. 본문 전개
...

## 3. 마무리
...
```

## 4. 성공 기준

- 예상 말하기 시간 기준 **5분 ± 20%** 분량:
  - 문단 수 / 글자 수 / 문장 수 등의 기준을 조합해 추정.
- 주제 설명이 명확하고, 과도한 선정성/허위 정보/혐오 표현이 없다.
- 도입–전개–정리의 기본 구조를 따른다.

## 5. 검증과 실패 처리

- **검증**
  - 규칙 기반:
    - 분량(길이) 검사
    - 섹션/헤더 존재 여부
  - LLM 자기 검증:
    - 주제와의 관련성
    - 정보 왜곡/과도한 자극성/민감 이슈 다루는 방식 점검
- **실패 처리**
  - 길이/구조 문제 → 자동 재시도 (최대 2회)
  - 내용적 위험(정치/민감 이슈, 명백한 허위 가능성 등) → 에스컬레이션 플래그를 남기고 사람 검토 필수로 표시.
