# video-assembler 스킬 (개요 버전)

## 1. 역할

- `script_{n}.md`, `images_{n}`, (옵션) TTS 오디오를 입력으로 받아,
  정해진 템플릿에 따라 5분짜리 영상 초안을 합성한다.
- 장면 전환, 간단한 텍스트 오버레이, 배경음악/나레이션 결합 등 최소한의 편집을 수행한다.

## 2. 입력

- `/output/{date}/script_{n}.md` 경로
- `/output/{date}/images_{n}/images_meta.json` 경로
- (옵션) `/output/{date}/audio_{n}.wav` 또는 TTS 설정

## 3. 출력

- `/output/{date}/video_{n}.mp4`

## 4. 성공 기준

- 영상 파일이 정상적으로 생성되고, 포맷/코덱이 유효하다.
- 영상 길이가 **5분 ± 20%** 범위 내에 존재한다.

## 5. 검증과 실패 처리

- **검증**
  - 파일 존재 여부, 기본 메타데이터(길이/해상도/프레임레이트)를 FFmpeg 등으로 확인.
- **실패 처리**
  - 합성 실패 → 템플릿을 단순화(예: 고정 배경 + 텍스트)하여 1회 재시도.
  - 재시도에도 실패하면 해당 영상은 실패로 표시하고, 다른 영상 처리 계속 진행.
